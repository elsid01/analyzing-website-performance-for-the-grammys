# Analyzing Website Performance for The Grammys

## Project Overview

This project analyzes website performance data from Grammys.com and RecordingAcademy.com to evaluate the impact of a website split that occurred on February 1, 2022. Prior to the split, both Grammy Awards and Recording Academy content were hosted on the same website. After the split, Grammys.com focused exclusively on Grammy-related content while RecordingAcademy.com hosted Recording Academy content separately.

The objective of this analysis is to determine whether separating the websites improved user engagement by examining traffic patterns, bounce rates, pages per session, average session duration, and mobile user behavior.

---

## Business Problem

The Recording Academy wanted to understand whether separating Grammys.com from RecordingAcademy.com improved the user experience and increased visitor engagement.

Key questions include:

- How does website traffic change around major Grammy events?
- How much traffic is generated during Grammy Awards show nights?
- Did user engagement improve after the website split?
- What percentage of visitors access the website through mobile devices?
- Should the websites remain separate or be merged back together?

---

## Dataset Description

The project uses two primary datasets:

### Grammys Dataset

Contains website analytics data for Grammys.com, including:

- Date
- Visitors
- Sessions
- Pageviews
- Bounced Sessions
- Average Session Duration
- Award Ceremony Indicators

### Recording Academy Dataset

Contains website analytics data for RecordingAcademy.com following the website split.

### Mobile Visitors Dataset

Contains daily mobile visitor counts used to analyze device usage trends.

## Methodology

### Part 1: Traffic Analysis

- Created a daily website traffic trend visualization.
- Identified major traffic spikes and linked them to Grammy-related events.
- Compared average visitor counts on award ceremony days versus regular days.
- Classified website content periods before and after the website split.

### Part 2: Engagement Analysis

Calculated and compared:

- Pages Per Session
- Bounce Rate
- Average Time on Site

Metrics were analyzed across:

- Grammys + Recording Academy period
- Grammys-only period
- Recording Academy website

### Part 3: Mobile User Analysis

- Used XLOOKUP to merge mobile visitor data.
- Calculated the percentage of visitors using mobile devices.
- Evaluated how mobile usage may influence engagement metrics.

### Part 4: Business Recommendation

- Interpreted findings from traffic and engagement analyses.
- Developed recommendations for The Recording Academy regarding the website split strategy.

## Key Findings

### Traffic Analysis

- Traffic spikes were strongly associated with major Grammy-related events.
- Grammy Awards show nights generated significantly higher visitor counts than regular days.

### Engagement Analysis

- Engagement metrics were compared before and after the website split.
- Pages per session, bounce rate, and average session duration were evaluated to determine user behavior changes.

### Mobile Analysis

- Mobile visitors represented a significant portion of website traffic.
- Mobile user experience likely influences overall engagement metrics.

---

## Business Recommendation

Based on the analysis, recommendations were developed regarding whether The Recording Academy should:

- Maintain separate websites,
- Reintegrate the websites,
- Or adopt an alternative content strategy.

The final recommendation is supported by quantitative engagement metrics and traffic analysis.
